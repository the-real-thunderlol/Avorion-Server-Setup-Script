# Avorion-Server-Setup-Script
A simple copy paste script designed to run the Server for the game Avorion.

The whole point of this script is to copy this onto your Debain/Ubuntu server and paste it as a bash file. Edit the variables that I've placed on the top of the file and then run it.
Your server will begin. 

- creates a user "steam" so its hosted as a user with no admin previlages
- If it crashes it will auto restart.
- Self hosting.
- Basically saving your time instead of you manually doing the whole thing
